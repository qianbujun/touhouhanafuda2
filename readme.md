# 东方札 Touhouhanafuda
一款以《东方Project》为题材、结合花札机制的原创牌类游戏。

## 🎴 简介
东方札 是一个以《东方Project》系列角色与场景为基础的卡牌游戏，灵感来源于传统日本游戏「花札」。玩家通过收集特定组合的卡牌获得分数，策略与运气兼具。

目前 2.0 版本已全面升级为 Web 网页版，支持在线联机与人机对战。

## 🎮 游玩方式与操作逻辑

**在线游玩**
目前可在 https://thfuda.panaolin.com/ 直接游玩。

**操作逻辑**
为了提升出牌的手感和防止误触，Web 版的操作进行了如下设计：
- **查看匹配牌**：将鼠标滑过手牌，场上能与之匹配（吃掉）的公共牌就会**立刻高亮**，或者悬停或点击场上的公共牌，也可以高亮自己的手牌中可以配对的牌（必须点击手牌才能打出），方便玩家决策。
- **选中手牌**：在某张手牌上**悬停满 0.3 秒**，或直接**单击**该手牌，（建议电脑使用悬停，手机使用单击）它会向上浮起并置顶，进入“选中”状态。
- **打出手牌**：在手牌处于“选中”（上浮）状态时，**再次单击**该手牌即可将其打出（双击确认防误触）。如果不想打出，点击或悬停其他牌即可取消。
